# Research & Experiments

This directory contains experimental code, analysis scripts, and one-off tools that are not part of the production pipeline.

- `analysis/`: Notebooks and analysis scripts.
- `tools/`: Ad-hoc utilities.
- `audit/`: Verification scripts preserved for audit.
- `oneoff/`: Retired or one-time use scripts from `submit/` and `scripts/`.

NOT FOR PRODUCTION USE.
